## Codebase Patterns
- Project is generated with **XcodeGen**: `project.yml` is the source of truth. After adding/removing source files or build settings, run `xcodegen generate`. The `.xcodeproj` is gitignored (regenerate it).
- Build: `xcodebuild build -project HolisticHealth.xcodeproj -scheme HolisticHealth -destination 'platform=iOS Simulator,name=iPhone 16,OS=18.3.1' -derivedDataPath build`
- Test: same command with `test` instead of `build`. Pin `OS=18.3.1` on the iPhone 16 destination — the bare name is ambiguous (many OS runtimes installed).
- Sim verify: `xcrun simctl boot "iPhone 16"; xcrun simctl install "iPhone 16" <app>; xcrun simctl launch "iPhone 16" com.holistichealth.app; xcrun simctl io "iPhone 16" screenshot /tmp/x.png`. Built app is under `build/Build/Products/Debug-iphonesimulator/HolisticHealth.app`.
- Source layout: `HolisticHealth/App` (entry + root), `HolisticHealth/Features/<Feature>`, `HolisticHealth/Shared` (reusable views), `HolisticHealth/Resources` (Assets.xcassets). Tests in `HolisticHealthTests/`.
- Bundle id: `com.holistichealth.app`. Swift 5 language mode (avoids strict-concurrency churn during rapid build). iOS 17 deployment target.
- Tabs are defined by the `AppTab` enum in `App/RootView.swift` — order there is the bottom-bar order. Settings is a sheet, NOT a tab.
- Info.plist is auto-generated (`GENERATE_INFOPLIST_FILE: YES`); set Info.plist keys via `INFOPLIST_KEY_*` build settings in project.yml (camera + photo usage strings already added).
- **Design system** lives in `Shared/DesignSystem`. Use `Theme.Colors.*` (semantic) and `Theme.Typography.*`, never raw colors/fonts. Components: `FramedCard`, `InkPanel`, `.buttonStyle(.decoPrimary/.decoSecondary)`, `Chip`, `StatusTag`, `DecoTextField`, `OrnamentalRule`, `PhotoPlaceholder`, `SectionHeader`, `EyebrowText`. Wrap screens in `.decoBackground()`.
- Fonts are iOS-native: `Futura` (display/poster) + `Avenir Next` (body) — mapped in Typography.swift. No bundled fonts. For gold TEXT on paper use `Theme.Colors.accentText` (goldDeep) for contrast; plain `accent` gold is decorative only.
- Nav/tab bar appearance is set globally in `ThemeAppearance.apply()` (called from `HolisticHealthApp.init`). `Color(hex:)` helper exists for porting design-guide hex values.
